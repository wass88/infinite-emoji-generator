# infinite-emoji-generator

壁紙群（Wallpaper Groups）の対称性で Twemoji をページ全体にタイリングする Web アプリ。

## 開発

```bash
npm install
npm run dev
```

## ビルド & デプロイ

```bash
npm run build
# GitHub Actions で GitHub Pages に自動デプロイ
```

## 技術スタック

| 項目 | 技術 |
|------|------|
| フレームワーク | Vite + React + TypeScript |
| 描画 | Canvas API |
| 絵文字 | Twemoji |
| ホスティング | GitHub Pages |
